# Task 3 – Interactive Dashboard

## Objective

The objective of this task is to create an interactive business dashboard using Microsoft Power BI. The dashboard provides meaningful insights into sales performance, profit, and regional distribution using visualizations and KPI cards.

---

## Tools Used

* Microsoft Power BI Desktop
* CSV Dataset
* Microsoft Windows

---

## Dataset

The dashboard uses the cleaned sales dataset created in **Task 1 (Data Wrangling)**.

Dataset Fields:

* Order ID
* Order Date
* Customer Name
* Region
* Category
* Sub Category
* Sales
* Profit
* Quantity

---

## Dashboard Visualizations

The dashboard contains the following visualizations:

### 1. Sales by Category

* Visualization: Clustered Column Chart
* Purpose: Compare sales across different product categories.

### 2. Profit by Region

* Visualization: Pie Chart
* Purpose: Show the profit contribution of each region.

### 3. Sales by Region

* Visualization: Clustered Bar Chart
* Purpose: Compare sales performance across regions.

### 4. Total Sales

* Visualization: Card
* Purpose: Display the total sales value.

### 5. Total Profit

* Visualization: Card
* Purpose: Display the total profit value.

### 6. Region Slicer

* Visualization: List Slicer
* Purpose: Filter all dashboard visuals based on the selected region.

---

## Key Insights

* Technology category generated the highest sales.
* Furniture recorded the second-highest sales.
* Office Supplies generated the lowest sales.
* West region contributed the highest profit.
* South region contributed the lowest profit.
* Interactive filtering allows users to analyze sales and profit by region.
