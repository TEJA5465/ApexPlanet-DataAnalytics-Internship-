# Task 5 – Portfolio & GitHub Submission

## Objective

The objective of this task is to create a professional portfolio showcasing all internship work completed during the ApexPlanet Data Analytics Internship. The portfolio is organized in a GitHub repository with proper documentation and supporting files for easy access and evaluation.

---

## Project Overview

This portfolio contains all the completed internship tasks, including data cleaning, exploratory data analysis, interactive dashboards, and data storytelling. It also includes the final presentation, GitHub repository links, LinkedIn post content, certificates, and supporting resources.
