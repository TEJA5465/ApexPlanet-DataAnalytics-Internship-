# Task 3 – Interactive Dashboard

## Objective

The objective of this task is to design and develop an interactive business dashboard using Microsoft Power BI. The dashboard transforms the cleaned sales dataset into meaningful visualizations that help analyze business performance and support data-driven decision-making.

---

## Tools Used

* Microsoft Power BI Desktop
* Microsoft Excel / CSV Dataset
* Windows Operating System

---

## Dataset

The dashboard is built using the cleaned sales dataset prepared during **Task 1 – Data Wrangling**.

The dataset contains the following fields:

* Order ID
* Order Date
* Customer Name
* Region
* Category
* Sub Category
* Sales
* Profit
* Quantity

---

## Dashboard Visualizations

The Power BI dashboard includes the following visualizations:

### Sales by Category

Displays total sales for each product category using a clustered column chart.

### Profit by Region

Displays profit contribution from each region using a pie chart.

### Sales by Region

Compares regional sales performance using a clustered bar chart.

### Total Sales

Displays the total sales value using a KPI Card.

### Total Profit

Displays the total profit value using a KPI Card.

### Region Slicer

Allows users to filter the entire dashboard based on the selected region.

---

## Dashboard Features

* Interactive visualizations
* Region-based filtering
* Business KPI cards
* Category-wise sales comparison
* Region-wise profit analysis
* Easy-to-understand business insights
